⚙️ Вариативная часть

Вариативная часть представляет собой исследовательскую и инженерную дейтельность, посвящённую созданию простого 3D-рендерера на Java с использованием библиотеки Swing. Она стала площадкой для изучения работы с матрицами трансформации, алгоритмами рендеринга, z-буфером и отрисовкой полигона через треугольники.

В ходе реализации были добавлены дополнительные функции: вращение по трём осям, управление мышкой, отображение локальных осей и сглаживание модели. Этот проект позволил глубже понять базовые 3D-принципы, используемые в игровых движках.
